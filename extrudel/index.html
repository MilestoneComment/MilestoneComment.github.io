What's new in v2.1

Now does a whole lot more.

Lithophane modes — Flat Panel, Cylinder / Lampshade, Window Hanger, Nightlight

Generate smooth height-map STLs directly from greyscale images. Every mode has its own geometry controls — arc angle, wrap count, base depth, notch slot, and more.

Window Hanger through-holes

Add two hanging holes in a single click. Diameter, inset, and side-offset are all adjustable. The holes are punched as real through-geometry — watertight and slicer-ready.

Nightlight base — flat and arc-shaped

Flat nightlights get a rectangular base with a U-slot for the plug. Curved nightlights get an arc-shaped base whose front edge follows the panel exactly, with an optional plug slot in the back wall.

English · Español · 简体中文 — live language switching

Switch language from the Settings menu and every label, button, error message, and the 3D viewer updates instantly. No restart needed. Choice is remembered across sessions.

mm / in toggle, default folders, window mode — all persistent

Switch between millimetres and inches any time — your choice is now remembered across launches. Set a default input and output folder so Browse always opens in the right place. App can start maximised, fullscreen, or windowed.

Denoise input — smarter photo handling

New optional checkbox in the lithophane settings. When ticked, Extrudel applies a small Gaussian blur to the source image before building geometry — knocking out JPEG artefacts and scanner grain without over-smoothing the result. Leave it off for clean SVGs and line art.

Sharper defaults

Default Smoothing is now 1 (was 2), and the hidden pre-blur that previously ran silently on every flat lithophane is removed. Out-of-box output is noticeably crisper for line art and stencils. Photo users have two knobs — Smoothing and Denoise input — to dial in exactly the softness they want.

Circle mode no longer clips corners

Previously, filled pixels in image corners could stick outside the circle boundary. Extrudel now finds the farthest solid pixel and shrinks the scale so everything fits inside.

How it works

From image to STL
in five steps.

1

Drop your image

Drag a PNG, JPG, BMP, TIFF, or WEBP onto the window — or click Browse. Colour images are automatically converted to greyscale.

2

Choose a mode

Lithophane for height-map prints (flat, cylinder, window hanger, nightlight). Circle or Rectangle extrusion for stencil-style solid prints.

3

Enter your dimensions

Type in millimetres or inches — switch units any time and all fields convert. Extrudel scales the geometry to match exactly.

4

Tweak and preview

Adjust gamma, smoothing, thickness, and other mode-specific settings. The built-in 3D viewer shows the result before you save.

5

Convert and open in your slicer

Hit Convert to STL, choose a save location, and drop the file into Bambu Studio, PrusaSlicer, Cura, or any other slicer.

Add two properly-punched hanging holes to any Window Hanger. Watertight geometry — no slicer errors.

Output is written in binary format via NumPy — roughly 10× smaller than ASCII STL and opens instantly in any slicer.
