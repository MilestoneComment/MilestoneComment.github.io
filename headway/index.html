Configurable Defaults

Set your own default browse location so file dialogs always open where you work.

How it works

Up and running in four steps.

1

Select your target folder

Point Headway at any folder containing .docx files. Subfolders are not processed, keeping things predictable.

2

Pick a master document

Choose one .docx whose header Headway will copy into every file in your folder.

3

Choose your options

Update headers, enforce even page counts, or run both in a single pass.

4

Run and review the log

Headway processes every file and shows a real-time log. Backups are created automatically.
